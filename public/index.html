URL Shortener

How It Works

Pass in a new URL to the address bar using the new path, and a shortened URL will be returned
Enter this shortened URL and it will redirect you to your original site

Example Usage:

https://url-shortener-me.herokuapp.com/new/https://www.google.com

Example Output:

{ "original_url":"http://foo.com:80", "short_url":"https://url-shortener-me.herokuapp.com/HJ9UXNAde" }

In Other Words:

This Link:

https://url-shortener-me.herokuapp.com/HJ9UXNAde

Will Redirect To:

https://www.google.com/
